# 🎓 Academic Result Portal

## 📌 Overview

Academic Result Portal is a web-based application developed using PHP and MySQL that allows students to securely access and manage their academic results. The system supports PDF-based result uploads, automatic data extraction, and Excel export functionality.

---

## 🚀 Features

* 🔐 Student Login System (Session-based authentication)
* 🔍 Search Results using Student ID
* 📄 Upload PDF and Automatically Extract Data
* 🗄️ Store Results in MySQL Database
* 📊 Download Class Results in Excel Format
* 🎨 Modern UI using Bootstrap

---

## 🛠️ Tech Stack

* PHP
* MySQL
* HTML / CSS
* Bootstrap
* PhpSpreadsheet
* Smalot PdfParser

---

## ⚙️ Setup Instructions

1. Install XAMPP
2. Start Apache & MySQL
3. Place project folder in `htdocs`
4. Create database: `ResultManagement`
5. Import required tables
6. Run project on:
   http://localhost/result-management
